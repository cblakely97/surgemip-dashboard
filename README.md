# SurgeMIP Dashboard

Static website showing progress and validation results for the SurgeMIP global storm surge simulations.

## Local preview

The site uses `fetch()` to load JSON data, which requires HTTP (not `file://`). Run a local server:

```bash
cd surgemip-dashboard
python3 -m http.server 8000
```

Then open http://localhost:8000.

## Updating data

Data is generated by scripts in the [STOFS2D-Global_SurgeMIP](https://github.com/cblakely97/STOFS2D-Global_SurgeMIP) repo:

```bash
# Update scenario progress bars
python postprocessing/update_progress.py

# Extract station time series and GESLA comparisons
python postprocessing/extract_station_data.py \
  --years 1979 \
  --gesla-dir /path/to/GESLAv4/ \
  --adcirc-dir /path/to/output/ \
  --rotm /path/to/fort.rotm \
  --output-dir /path/to/surgemip-dashboard/data/
```

## Pages

- **Progress** (`index.html`) — completion status for each scenario (CFS reanalysis, HadGEM present/future)
- **Station Comparison** (`comparison.html`) — interactive map of GESLA tide gauge stations with ADCIRC vs observation time series
